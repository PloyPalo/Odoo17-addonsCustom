Installation Guide

  • Copy module to Odoo addons path
  • Restart Odoo server
  • Update Apps list
  • Install "Database Backup Manager"
  • Grant permissions to users who need access by adding them to the group "Database Backup Manager"

    • User Manual

    Backup Process

  • Go to Settings -> Technical -> Database Management -> Backup Database
  • Click "Create" to create new backup configuration
  • Fill in required information:
    • Name: Configuration name
    • Database Name: Select database to backup
    • Backup Directory: Specify backup location
    • Backup Type: Choose Manual or Scheduled
  • For Scheduled backup, configure:
    • Schedule Interval: Daily/Weekly/Monthly
    • Backup Time: Time to perform backup
  • Click "Save" and "Backup Now" for immediate backup

    • Restore Process

  • Go to Settings -> Technical -> Database Management -> Restore Database
  • Click "Create" to start new restore process
  • Fill in required information:
    • Description: Restore description
    • Master Password: Odoo master password
    • New Database Name: Name for restored database
    • Server Backup File: Select backup file to restore
  • Click "Restore Database"
  • Wait for process to complete and restart Odoo server
    • Note: Always ensure you have sufficient disk space before performing backup or restore operations.
    Important: Backup your existing data before performing any restore operations.

    คุณสมบัติหลัก

    การจัดการสำรองข้อมูล

    • สำรองฐานข้อมูลด้วยตนเอง
    • ตั้งเวลาสำรองอัตโนมัติ
    • ตัวเลือกการกำหนดเวลาที่ยืดหยุ่น
    • กำหนดโฟลเดอร์สำรองข้อมูลเอง

    ความสามารถในการกู้คืน

    • กู้คืนฐานข้อมูลได้ง่าย
    • ตรวจสอบไฟล์สำรองอัตโนมัติ
    • สร้างฐานข้อมูลใหม่
    • กระบวนการกู้คืนที่ปลอดภัย

    ระบบความปลอดภัย

    • ควบคุมการเข้าถึงตามบทบาท
    • ป้องกันด้วยรหัสผ่านหลัก
    • จัดการไฟล์อย่างปลอดภัย
    • บันทึกและจัดการข้อผิดพลาด

    คู่มือการติดตั้ง

  • คัดลอกโมดูลไปยัง addons path ของ Odoo
  • รีสตาร์ท Odoo server
  • อัพเดทรายการแอพ
  • ติดตั้ง "ระบบจัดการสำรองฐานข้อมูล"
  • กำหนดสิทธิ์ให้ผู้ใช้ที่ต้องการใช้งาน โดยเพิ่มเข้ากลุ่ม "Database Backup Manager"

    • คู่มือการใช้งาน

    ขั้นตอนการสำรองฐานข้อมูล

  • ไปที่ Settings -> Technical -> Database Management -> Backup Database
  • กดปุ่ม "Create" เพื่อสร้าง backup configuration ใหม่
  • กรอกข้อมูลที่จำเป็น:
    • Name: ชื่อการตั้งค่า
    • Database Name: เลือกฐานข้อมูลที่ต้องการสำรอง
    • Backup Directory: ระบุที่เก็บไฟล์สำรอง
    • Backup Type: เลือก Manual หรือ Scheduled
  • กรณีเลือก Scheduled ให้กำหนด:
    • Schedule Interval: รายวัน/สัปดาห์/เดือน
    • Backup Time: เวลาที่จะทำการสำรอง
  • กดปุ่ม "Save" และ "Backup Now" เพื่อทำการสำรองทันที

    • ขั้นตอนการกู้คืนฐานข้อมูล

  • ไปที่ Settings -> Technical -> Database Management -> Restore Database
  • กดปุ่ม "Create" เพื่อสร้าง restore configuration ใหม่
  • กรอกข้อมูล:
    • Description: คำอธิบายการกู้คืน
    • Master Password: รหัสผ่าน master ของ Odoo
    • New Database Name: ชื่อฐานข้อมูลใหม่
    • Server Backup File: เลือกไฟล์สำรองที่ต้องการกู้คืน
  • กดปุ่ม "Restore Database"
  • รอจนกระบวนการเสร็จสิ้นและรีสตาร์ท Odoo server
    • หมายเหตุ: ตรวจสอบพื้นที่ว่างในดิสก์ให้เพียงพอก่อนทำการสำรองหรือกู้คืนข้อมูล
    สำคัญ: ควรสำรองข้อมูลที่มีอยู่ก่อนทำการกู้คืนใดๆ
